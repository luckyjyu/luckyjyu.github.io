练习

压缩包管理

  1. Gzip命令能够对目录进行压缩吗？

    No

  2. Gzip命令对对文件压缩完毕之后会保留原文件吗，如果能请写出该命令？

    No

    Gzip ./*.txt

    .gz格式

  3. Gzip命令对多个文件压缩之后会生成一个压缩包还是多个，都是什么格式的？

    多个，xxx.gz

  4. Bzip2命令能够对目录进行压缩吗？

    No

  5. Bzip2命令对对文件压缩完毕之后会保留原文件吗，如果能请写出该命令？

    可以

    Bzip2 -k ./*.txt

  6. Bzip2命令对多个文件压缩之后会生成一个压缩包还是多个，都是什么格式的？

    多个，xxx.bz2

  7. 使用tar命令将目录 dir1压缩成gz格式，请写出该命令

    tar zcvf mydir.tar.gz dir1

  8. 使用tar命令对压缩包test.tar.bz2 进行解压缩， 请写出该命令

    tar jxvf test.tar.bz2

  9. 使用tar命令将压缩包test.tar.gz解压到家目录下，请写出该命令

    tar zxvf test.tar.gz -C /home

  10. 使用rar命令压缩目录mytest，写出该命令

    rar a myrar mytest（无需指定压缩包的后缀，会自动添加）

    可以在后面加上-r（不加也可以）

  11. 使用rar命令解压缩 mytest.rar，写出该命令

    解压到当前目录：rar x mytest.rar

    解压到指定目录：rar x mytest.rar + 解压路径

  12. 使用zip命令压缩目录mytest，写出该命令

    zip -r myzip mytest

  13. 写出命令，解压缩mytest.zip

    unzip mytest.zip -d + 解压目录

压缩：命令+参数+压缩文件的名字+压缩的文件或目录

解压缩：命令+压缩包名字+参数（rar没有参数）+解压目录

  1. 查看当前在线用户的状况的命令

    who

  2. Linux下如何切换设备终端？

    一共有7个设备终端，前六个是文字终端，第七个是界面终端

    Alt+Ctrl+f1-f7

  3. Linux下各个设备终端之间是相互依赖的还是互补影响？

    互不影响，可认为是七台电脑，各司其职

  4. 写出命令，在终端下查看操作系统下所有的bash进程

    ps aux|grep “bash”

  5. 如何使用kill命令查看信号编号？

    kill -l

  6. 写出命令，杀死进程编号为998877（PID）的进程

    kill -9 998877

  7. 查看当前进程环境变量的命令是哪一个？

    evn —–PATH

    env|grep PATH

ftp服务器搭建

  1. ftp服务器安装完毕之后，服务会随系统的启动而自动启动吗？

    会

  2. 如果需要修改ftp服务器的配置文件，应该去哪个目录下找？

    /etc/vsftpd.conf

  3. ftp服务器配置文件修改完毕之后，修改的配置能够马上生效吗，为什么？

    不会

    需要重启，不重启系统不会自动去读修改的配置文件

  4. 假设使用的ftp服务器为vsftpd，写出服务器重启命令？

    sudo service vsftpd restart

  5. 从ftp服务器下载文件的命令是什么，上传文件的命令是什么？

    下载：get

    上传：put

  6. Fpt服务器自带的客户端能够对目录进行上传和下载操作吗？

    不能，打包可以上传

  7. 使用ftp自带的客户端如何使用匿名用户登录服务器，写出命令

    ftp serverIP

    用户名：anonymous

    密码：回车

  8. 登录到ftp服务器之后，使用什么命令能够退出登录？

    quit/exit/bye

  9. 如何使用lftp客户端匿名登录到ftp服务器？

    lfpt + serverIP

    login

  10. 使用lftp如何上传下载文件，写出命令

    上传：put

    下载：get

  11. 使用lftp如何上传下载多个文件，写出命令

    上传多个：mput xxx xxx

    下载多个：mget xxx xxx

  12. 使用lftp如何上传下载目录，写出命令

    上传目录：mirror dir

    下载目录：mirror -R dir

nfs网络共享服务器

  1. Nfs服务器配置文件修改完毕之后，会马上生效吗？

    不行，需要重启服务

  2. Nfs服务器安装成功之后，服务会随系统自动启动吗？

    会的，否则重启命令为啥叫restart呢

  3. 使用什么命令能够重启nfs服务器？（服务器名 nfs-kernel-server）

    sudo nfs-kernel-server restart

  4. 客户端如何登陆到nfs服务器，写出命令

（ip：192.168.1.100，共享目录：/home/itcast/nfs）
