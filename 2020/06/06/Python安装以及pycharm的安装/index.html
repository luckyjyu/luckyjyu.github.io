一、Python3.8安装

1、下载Python安装包【下载地址】

选择Downloads–>Windows

选择Windows x86-64 web-based installer 版本

2、安装python3.8.3

​ 双击exe文件进行安装，选择自定义安装，其他选择不变，更改安装路径即可，勾线Add Python 3.8 to PATH，否自后面需要自行添加路径

3、验证是否安装成功

在cmd中输入python，若出现版本号等信息则表示安装成功，如图所示
